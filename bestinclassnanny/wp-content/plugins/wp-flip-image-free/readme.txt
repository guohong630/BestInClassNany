== Installation ==

= See the Screenshots for detailed Installation and settings. =
* Install and activate the plugin using the "Plugins" -> "add new" menu in WordPress.
* Go to Flip Image Free menu in WordPress.  
* Choose gallery style, upload images, set text and URL, click "Save Changes".
* Copy and paste the shortcode [WP-Flip-Image-Free] on any necessary page or post.
